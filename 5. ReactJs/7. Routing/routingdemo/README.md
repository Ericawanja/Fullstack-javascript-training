# React Routing demo
In this lesson, we created a simple store that has home, product and single product page to demonstrate routing 

Concepts Covered
- setting up the routing
- handling navigation
- dynamic routes
- nested routes
- navigation components
- useRoutes

## To check the demo
- Fork the repo
- clone the repo
- navigate to the directory
- run `npm install`
- run `npm start`
